Why am I in this class?

Being a computer science profesisonal, I have to work and collaborate with others on a regualr basis. Whether I am working on a code reviews at the office or attending an online video lecture at college, Collaborative Softwares have become an integral part of my life. Collaborative Softwares are different from other single-user softwares in a sense that there is an addition social challenges and complexities introduced due the the group dynamics of the users working collaboratively. And, as an aspiring software developer, I believe it is important for me to know these challenges and how to tackle them. Therefore, I have joined this course to get familiar with the history of these collaborative softwares and to understand more about designing such kind of softwares.

What features of these Software Tools do I think help me the most with regards to working with others?

Ease of Use

The most common thing about these softwares is that they are easy to setup and we can start usign them very quickly. If I want to discuss something with some other colleague, then it is just "one-click" away (given their availability :P).

Rich User Experience

Most of them have very simple, clean and intuitive UI which results in a very rich and productinve user experience. For example, Google Docs is a collaborative real-time document sharing platform. Despite providing lots of features, the developers have tried to keep a minimalistic UI by providing drop downs/on-click sliders rather than overloading the screen with buttons.

How I think Collaborative Software Tools can be improved?

Dependency on Fast internet service.

Most of the collaborative softwares webapps require a fast internet connection and the UI elements, that frontend framework use, have become heavier (larger in size). I would suggest these websites to use lighweight UI elements.

Security concerns with data collection.

These collaborative softwares collect lots of data related to user's activity, location etc. in order to provide good collaborative experience. There is alwasys a risk involved in collecting and stroring this kind of sensitive data.
